考虑密码如果直接明文写在这条命令行上，有些不方便（怕被别人看到），可以只输入：mysql -u 用户名 -p 然后回车，此时提示你输入密码，这时候输入的密码就不再是明文的了。

使用PHP脚本创建数据库
PHP 使用 mysqli_query 函数来创建或者删除 MySQL 数据库。
该函数有两个参数，在执行成功时返回 TRUE，否则返回 FALSE。(bool布尔型）

connection 是必须写的，规定要使用的MySQL连接。
query 是必须写的，规定查询字符串。为了在数据库中寻找某一特定文件、网站、记录或一系列记录，由搜索引擎或数据库送出的消息。
mysql_query() 函数执行一条 MySQL 查询。且在query规定要发送的 SQL 查询。注释：查询字符串不应以分号结束。（不太明白）
