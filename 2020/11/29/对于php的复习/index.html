PHP
在 PHP 中，使用是通过预定义数组 $_SESSION 的存储和读取 session。
array_sum() 返回数组中值的和。
PHP提供了 setcookie() 函数来设置 cookie。
$_SERVER ——这是一个包含诸如头、路径和脚本位置等信息的数组。这个数组中的条目由web服务器创建，不能保证每个 web 服务器都能提供这些功能。
array_reverse() 以相反的顺序返回数组。
一旦使用 fopen() 函数打开文件，就可以使用一个名为 fread() 的函数读取它。这个函数需要两个参数，要读取打开的文件和要读取的最大字节数。
session 可以由 session_destroy()函数销毁，也可使用使用 $_SESSION = array() 清空 $_SESSION 数组。
PHP 提供了一个 getenv() 函数来访问所有环境变量的值。
异常处理类方法 getMessage() 返回异常的消息。
数值数组－带有数字索引的数组，值以线性方式存储和访问。
索引数组是数字下标；关联数组是键值下标。
关联数组操作起来比较方便，而且信息量相对来说大些。
索引数组 系统自动指定索引,或者索引全为数字

关联数组 索引为非数字

PHP 数字索引数组一般表示数组元素在数组中的位置，它由数字组成，下标从 0 开始，数字索引数组默认索引值就是从数字 0 开始的，不需要特别指定，PHP 会自动为索引数组的键名赋一个整数值，然后从这个值开始自动增量，当然，也可以指定从某个位置开始保存数据。数组可以构造成一系列的“键-值（key-value）对”，其中每一对都是数组的一个项目或者元素（element）。对于数组中的每一个项目，都有一个与之关联的键（key）或者索引（index）相对应。
要从存储在字符串中的XML文档创建 SimpleXML 对象，请将该字符串传递给 simplexml_load_string()，它返回一个 SimpleXML 对象。
常量与变量不同的是，您不需要一个带 $。
只有标量数据(布尔值、整数、浮点数和字符串)可以包含在常量中。
通过 $this 引用类本身中的属性或方法。
require 一个文件存在错误的话，那么程序就会中断执行了，并显示致命错误。
include 一个文件存在错误的话，那么程序不会中断，而是继续执行，并显示一个警告错误。
is_a 如果对象属于该类或该类是此对象的父类则返回 TRUE。
PHP 对空格不敏感，这意味着一行中有多少空格字符并没啥关系，一个空白与许多空白符都是一样的。
date() 函数的作用是返回一个格式化的日期字符串。
$_REQUEST可以用来获取通过 PHP 中的 GET/POST 方法发送的信息。
PHP 提供了一个名为 __construct() 的特殊函数来定义构造函数，可以向构造函数传递任意多个参数。

PHP 提供 $_GET 关联数组来使用 GET 方法访问所有发送的信息。
htmlentities() 函数把字符转换为 HTML 实体。
curl_init 初始化一个 CURL 会话。
mysql_fetch_row() 函数从结果集中取得一行作为数字数组。
连接 MySql 数据库
mysql_connect(“localhost”,$username,$password)
